# Overview
- 소개: 널리는 전시회를 보다 널리 구경하자는 취지에서 지은 이름으로, 서울시 전시회의 활성화와 방문객의 편의를 목적으로 제작된 애플리케이션입니다.  
- 시연 영상: https://youtu.be/S6yopJr-VkQ
